# aizidongkongzhi
项目简介本项目针对电加热炉温度控制系统，完成阶跃响应实验数据处理、多算法系统辨识建模、PID 控制器参数整定与智能优化仿真全流程实验。通过对比传统辨识方法、最小二乘一阶 / 二阶惯性滞后模型，以及 Z-N 经验整定、差分进化 DE、粒子群 PSO 三种 PID 优化算法，完成模型精度校验与闭环控制性能量化分析.
1 实验对象与基础参数1.1 被控对象：电加热炉（热工滞后对象）
采样总点数：21601 组
采样时长：0 ~ 10800 s，采样间隔：0.5 s
阶跃输入电压：3.5 V（35% 加热功率）
初始温度：16.8488 ℃，稳态温度：51.3306 ℃
系统静态增益：K=ΔuΔy​=9.8519 ∘C/V
控制目标：炉膛温度稳定至设定值 SP=35∘C
